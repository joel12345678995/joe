# JoeFamily Treasury System

A production-ready treasury platform for families, SACCOs, church groups, and community circles.

## Features

- ✅ Member Management with Role-Based Access
- ✅ Savings Contributions Tracking
- ✅ Loan Management with Approvals
- ✅ Real-time Transparency Dashboard
- ✅ Expense Tracking
- ✅ Payment Verification System
- ✅ Migration Tools (CSV Import)
- ✅ Responsive Design

## Tech Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS
- **Backend:** Supabase (Auth, PostgreSQL, Storage)
- **Deployment:** Vercel

## Environment Variables

Create a `.env.local` file with:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
